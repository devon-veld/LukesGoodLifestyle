# Luke's Good Lifestyle

Static export of the Luke's Good Lifestyle website (dark-theme design).
Home · Shop · Results · Training · Admin dashboard · cart/checkout · legal —
a self-contained single-page app (renders via `support.js`). Mobile responsive.

## Run locally
Any static server, e.g.:
```bash
python -m http.server 8080
```
Then open http://localhost:8080

## Deploy (Netlify)
No build step — it's static. `netlify.toml` sets publish `.` and a SPA redirect.
Admin dashboard passcode-free demo at the "Admin login" footer link.
